# Como usar este repositório

## 1. Edite o README principal

Abra `README.md` e troque os placeholders por suas informações.

## 2. Edite as áreas

Cada área fica dentro de `areas/`.

Exemplo:

```txt
areas/lua/README.md
```

Use esse arquivo para contar sua experiência, o que está estudando e o que pretende construir.

## 3. Adicione projetos futuros

Cada área tem 8 pastas preparadas:

```txt
projeto-01
projeto-02
projeto-03
projeto-04
projeto-05
projeto-06
projeto-07
projeto-08
```

Entre em uma delas e edite o README.

## 4. Coloque imagens

Use:

```txt
assets/images/
```

## 5. Coloque vídeos

Use:

```txt
assets/videos/
```

No README do projeto, linke o vídeo ou coloque uma imagem de capa.

## 6. Privacidade

Não coloque arquivos sensíveis no repositório.

Para conteúdo privado, crie um repositório privado separado.
